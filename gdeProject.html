GDE - Game Development Engine

This is my own game engine and main side-project. It is actively being developed and adding more features each month.

The engine is built in C++ with the combination of some libraries like SDL, OpenGL... It is focused totally to 2D development and it is at least as powerful as SFML (by benchmarks). It can deploy to Windows, Linux, Mac, Android and iOS and nintendo switch is already in development. For full description go to Home of this webpage.
