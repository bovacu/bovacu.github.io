Simple 2D C++ game engine

Make your own applications and games with this engine, fully provided of the basic features. RDE supports Windows, Linux, MacOs, iOS and Android platforms, with more in the way!

Get Started Github repo

Main features

Built with C++

The engine is fully built in C++ and is the language used to develop within the engine.

ECS and OOP

ECS built-in support if needed.

2D Rendering

Flexible renderer supporting Textures, Sprite Sheets, multiple cameras and custom shaders.

Cross-platform

RDE supports Windows, Linux, MacOs, iOS, Android and consoles are in the way!

Custom UI

RDE offers a custom solution for UI, with the basic elements ready to use with custom events.

Sound System

Basic system to play sound effects and background music.

Scene System

Customizable scene system that can be loaded through JSON files (and editor in the way).

Prefab System

Create a complex object through JSON and instanciate it as many times as needed.

Physics System

Integrated thanks to Chipmunk and fully ready to work with custom events.

Particle System

Basic system for rendering particles in an efficent way.

In-Engine Console

RDE offers an internal console customizable with your own commands.

Free and Open Source

The engine is completely free, use it and modify it as you want.
