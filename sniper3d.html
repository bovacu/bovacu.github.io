Sniper3D

This project is in the top 3 most downloaded shooter ever, with more than 500 million downloads. Sniper3D is a game about killing enemies by contracts and testing your accuracy.


The project is developed in Unity and in this project I focused completely on the client side, fixing old and new bugs, fixing performance and bottle neck issues and also memory problems. In the last time I worked in the project also on some new features of the game and helped with the huge revamp.
