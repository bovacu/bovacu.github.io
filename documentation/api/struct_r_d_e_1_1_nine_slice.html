Detailed Description

Defines how the texture must be rendered so deformations on X and Y won't affect it's quality. It is separated 9 parts as follows: 

  ____|____________|____
 | 6  |     7      | 8  |
 |----|------------|----| top
 |    |            |    |
 | 3  |     4      | 5  |
 |    |            |    |
 |----|------------|----| bottom
 |_0__|_____1______|_2__|
      |            |
     left         right

Elements 0, 2, 6 and 8 doesn't get streched. Elements 3 and 5 get streched in Y axis. Elements 1 and 7 get streched in X axis. Element 4 gets streched in XY axis.
