class UIInteractable

Component that every UI element that wants to interact with events on the screen must include.

Public Functions

explicit UIInteractable(const NodeID &_nodeId)

Public Members

UDelegate<bool(NodeID, Canvas*)> interactionTrigger

Sets the function that will allow the system to know when any of the below callbacks should be triggered or not. Usually this function will be if the mouse is inside the rectangles bound of the UI element.

It is not set by default by the engine and MUST be provided by the end-user.

MDelegate<void(MouseCode)> onClick

Callback triggered when the mouse is clicked.

MDelegate<void()> onMouseEntered

Callback triggered when the mouse has just entered.

MDelegate<void()> onMouseExited

Callback triggered when the mouse has just exited.

MDelegate<void(Vec2F)> onScroll

Callback triggered when the mouse is scrolled.

MDelegate<void(KeyCode)> onKeyPressed

Callback triggered when a key is pressed.

MDelegate<void(ControllerButtons)> onGamepadButtonPressed

Callback triggered when a gamepad button is pressed.

MDelegate<void(int)> onMobileClick

Callback triggered when the mobile screen is touched.

int priority = 0

Order of execution of the different UI elements.
