class InputManager

This class allows the user to ask for any user input in a easy and accessible way.

Attention

All managers destroy all the resources before closing the application, it is not needed to be done by the end user.

Public Functions

void init(Engine *_engine, Window *_window)

This initiates all of the crucial elements of the input system.

Attention

This is not meant to be called by end-users.

Parameters

  • _engine – Engine

  • _window – Window

void destroy()

Destroys all input system elements.

Attention

This is not meant to be called by end-users.
