class FontManager

This class manages all the fonts of the applications. It loads, unloads and gets the fonts of memory.

Attention

All managers destroy all the resources before closing the application, it is not needed to be done by the end user.

Font *getSpecificFont(FileManager &_fileManager, const std::string &_fontName, int _fontSize)

his function tries to return the font with the specific font size. If the font size is near enough to one already loaded, it will return it. Otherwise it will create a new font with the desired size because if using one too far from the loaded fonts sizes, it will be blurry and bad rendered.

Parameters

  • _fontName – Font name

  • _fontSize – font size
