static void beginDraw(Camera &_camera, Transform *_cameraTransform)

Renders textures, sprites… It includes 2D batching for a much more performant drawing.

Parameters

  • _camera – It is the main camera of the scene (or other camera if a special drawing is needed).

  • _renderShader – The shader we want to use to render, by default the basic one.

static void beginDebugDraw(Camera &_camera, Transform *_cameraTransform, float _thickness = 1.f)

Renders geometry for debugging. It also includes batching.

Parameters

  • _camera – It is the main camera of the scene (or other camera if a special drawing is needed).

  • _renderShader – The shader we want to use to render, by default the debugging one.
