class Console

This is a class basically used for debugging and allows to create custom commands to execute on runtime.

Attention

All managers destroy all the resources before closing the application, it is not needed to be done by the end user.

Public Functions

void init()

Initiates all the crucial elements of the Console.

Attention

This function should not be called by end users.

template<auto Func, typename Class>
inline void addCommand(const std::string &_commandName, const std::string &_description, Class *_class = nullptr, const std::string &_argumentsDescription = "")

This function allows to create a custom command.

Parameters

  • _commandName – the name you will use to call the command.

  • _description – the description displayed if command ‘help’ is called.

  • _class – this is the instance of the class that contains the function to be called, usually just pass ‘this’.

  • _argumentsDescription – the arguments that should be passed that will be displayed if ‘help’ is called.
