class ShaderManager

TThis class is in charge of compiling and storing the references of all of the shaders.

Attention

All managers destroy all the resources before closing the application, it is not needed to be done by the end user.

Public Functions

void init(FileManager *_fileManager)

Initializes all the crucial elements of the Shader Manager.

Attention

This is not meant to be called by end-users.

void destroy()

Destroys all of the Shaders from the GPU.

Attention

This is not meant to be called by end-users.
